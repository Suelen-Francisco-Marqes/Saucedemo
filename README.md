# 🧪 Saucedemo - Automação de Testes

Projeto de automação de testes end-to-end do site SauceDemo, 
desenvolvido para praticar habilidades em QA e automação.

## 🚀 Tecnologias
- Cypress
- JavaScript
- GitHub Actions (CI/CD)

## ✅ Cenários cobertos
- Login
- Filtros do inventory
- Carrinho
- Checkout
- Resumo do checkout

## ⚙️ Como rodar
```bash
npm install
npx cypress open
```
